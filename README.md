# 📖 Overview

TableForge JS is a beginner-friendly JavaScript application that generates multiplication tables based on numbers entered by users.

The project demonstrates fundamental programming concepts including functions, loops, arithmetic operations, user input handling, and console output generation.

It serves as a practical introduction to algorithmic thinking and JavaScript development.

# 🔍 How It Works

1. User enters a number.
2. JavaScript receives the input.
3. A loop iterates from 1 to 10.
4. Multiplication results are calculated.
5. The complete table is displayed in the browser console.

Example Input:

```text
5
```

Output:

```text
5 × 1 = 5
5 × 2 = 10
5 × 3 = 15
...
5 × 10 = 50
```
